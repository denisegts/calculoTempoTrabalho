# calculoTempoTrabalho
Calculo de tempo de trabalho baseado em tarefas


Esse projeto utiliza Python e Streamlit para automatizar cálculos de produtividade bancária. 

Funcionalidades


. Cálculo Automatizado: Processamento de tempos baseados em três tipos de tarefas (5.91min, 7.59min e 0.85min).

. Gestão de Limite: Alertas visuais para jornadas que excedem o limite de 8h30min (510 minutos).

. Interface Intuitiva: Abas organizadas para calculadora, análise e configurações.

. Estado de Sessão: Botões para limpar campos e resetar cálculos instantaneamente.
